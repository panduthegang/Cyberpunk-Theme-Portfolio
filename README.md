## 🚀 Quick Start

1. 📥 **Clone the Portal**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. 🗺️ **Navigate to Base**
   ```bash
   cd portfolio
   ```

3. 🌐 **Launch Local Server**
   ```bash
   # 🐍 Python Method
   python -m http.server 8000
   
   # ⚡ Node.js Method
   npx serve
   ```
